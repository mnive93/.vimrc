.vimrc
======
Settings for vim editor 
The plugins used are : 
pathogen.vim: Manages the runtime path of the plugins
NERD Tree: A tree explorer plugin for vim
NeoComplete - Keyword completion system by maintaining a cache of keywords.
Color Scheme:
Vividchalk
