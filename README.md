.vimrc
======
Settings for vim editor 
The plugins used are : 
<ul>
<li>pathogen.vim: Manages the runtime path of the plugins</li>
<li>NERD Tree: A tree explorer plugin for vim</li>
<li>NeoComplete - Keyword completion system by maintaining a cache of keywords.</li>
</ul>
Color Scheme:
<ul>
<li>Vividchalk </li>
<li>Solarize </li>
</ul>
